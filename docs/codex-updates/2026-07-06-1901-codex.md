## Catatan Teknis

- Jangan tambah dependency untuk shell simple mode. Native React + Next routing sudah cukup.
- Jangan pindahkan semua logic lama sekaligus; cukup bungkus children dengan shell yang berbeda.
- Simple mode bukan berarti fitur dihapus dari database, hanya UX dibuat lebih pendek supaya UMKM tidak overwhelmed.
- Advanced mode tetap mempertahankan sidebar/hub karena cocok untuk Company dan operasional multi-divisi.
