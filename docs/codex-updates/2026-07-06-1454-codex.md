## Yang Dikerjakan
- Menambah jenis produk baru di Prisma:
  - `GOODS` untuk barang stok biasa.
  - `SERVICE` untuk jasa.
  - `ASSEMBLY` untuk produk rakitan/bundle/produksi yang tetap bisa stok.
  - `NON_INVENTORY` untuk item jual tanpa stok seperti ongkir/biaya layanan.
  - `COST` untuk item biaya internal/HPP/tenaga kerja.
- Menambah migration `20260706143000_add_advanced_product_kinds`.
- Memperbarui form produk:
  - Selector jenis produk menjadi Barang, Jasa, Rakitan, Non-Stok, Biaya.
  - Stok dan expired hanya tersedia untuk Barang/Rakitan.
  - Durasi layanan hanya untuk Jasa.
  - Item Biaya diberi penjelasan bahwa tidak tampil di POS.
- Memperbarui validasi server produk supaya jenis item baru aman.
- Memfilter item `COST` dari daftar produk kasir/POS.
- Memperbarui listing produk agar label jenis item terlihat.
