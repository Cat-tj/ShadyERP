# 2026-07-05 20:44 Codex Update

## Fokus

Menambahkan setup QRIS usaha dari Pengaturan Bisnis agar owner bisa scan QRIS statis sekali, lalu POS memakai QRIS tersebut untuk membuat QRIS dinamis per nominal transaksi.

## Perubahan

- Menambahkan field `staticQrisPayload` di `TenantSetting`.
- Menambahkan migration:
  - `prisma/migrations/20260705204007_add_static_qris_setting/migration.sql`
- Pengaturan Bisnis sekarang punya section `QRIS usaha`.
- Owner bisa:
  - buka kamera untuk scan QRIS statis,
  - upload foto QRIS sebagai fallback,
  - atau paste payload QRIS manual.
- Payload QRIS divalidasi sebelum disimpan.
- Kasir/POS sekarang mengambil QRIS dari pengaturan tenant.
- Input manual QRIS di payment sheet masih ada sebagai fallback per device.
