# 2026-07-05 20:03 Codex Update

## Fokus

Menambahkan workflow transfer stok antar cabang untuk mendukung operasional cafe multi-outlet.

## Perubahan

- Transfer stok tidak lagi hanya diperlakukan sebagai pemindahan instan.
- Menambahkan status transfer:
  - `REQUESTED`
  - `SENT`
  - `RECEIVED`
  - `REJECTED`
  - `CANCELLED`
- Menambahkan field audit pada `StockTransfer`:
  - pengirim request
  - pengirim stok
  - penerima stok
  - jumlah request
  - jumlah dikirim
  - jumlah diterima
  - waktu approve/kirim/terima/tolak/batal
- Membuat migration database:
  - `prisma/migrations/20260705195637_add_stock_transfer_workflow/migration.sql`
- Mengubah service inventory:
  - request transfer tidak mengubah stok
  - kirim transfer mengurangi stok outlet asal
  - terima transfer menambah stok outlet tujuan
  - reject transfer menutup request tanpa mengubah stok
- Mengubah halaman transfer stok:
  - form sekarang membuat request transfer
  - daftar riwayat menampilkan status
  - request bisa dikirim atau ditolak
  - transfer yang sudah dikirim bisa diterima dengan jumlah aktual
- Route yang terkena:
  - `/inventory/transfer-stok`
  - `/produk/transfer-stok`
