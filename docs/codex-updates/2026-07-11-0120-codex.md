# Update Codex - 11 Juli 2026 01:20

## Scanner kamera selalu aktif di Kasir

- Menambahkan toggle **Scanner selalu aktif** di layar Kasir.
- Saat aktif, kamera tampil sebagai panel kecil yang tidak menutup katalog maupun invoice.
- Barcode atau QR yang cocok dengan SKU langsung ditambahkan ke invoice dan kamera tetap menyala untuk scan berikutnya.
- Menambahkan cooldown 1,5 detik untuk barcode yang sama agar satu produk tidak masuk berulang ketika kamera masih diarahkan ke labelnya.
- Tombol kamera lama tetap membuka mode scan sekali pakai.

## Catatan teknis

- Scanner berbasis Web `BarcodeDetector`, sehingga membutuhkan browser yang mendukung API tersebut dan izin kamera.
- Scanner USB/Bluetooth tetap didukung melalui input pencarian dan tombol Enter.
