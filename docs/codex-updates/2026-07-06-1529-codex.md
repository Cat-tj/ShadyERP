# 2026-07-06 15:29 - Codex

## Konteks
- User ingin akun admin aplikasi/platform (`Superadmin`) untuk mengelola client.
- User meminta tab `Modul` di akun toko disembunyikan, karena fitur client harus di-enable dari Superadmin, bukan dari admin toko.

## Yang Dikerjakan
- Menghapus tab `Modul` dari tab Pengaturan akun toko.
- Route `/pengaturan/modul` sekarang redirect ke `/pengaturan/bisnis`.
- Server action lama pengaturan modul tenant tidak lagi mengubah data dan mengembalikan pesan bahwa modul dikelola Superadmin.
- Menambah kontrol modul di halaman `/superadmin`:
  - Setiap tenant/client punya tombol `Modul`.
  - Superadmin bisa melihat jumlah modul aktif.
  - Superadmin bisa enable/disable modul fitur per tenant.
  - Simpan modul langsung update `Tenant.disabledModules`.
- Mengubah copy guard modul nonaktif agar owner toko tidak diarahkan ke tab Modul lagi.

## Catatan
- Tidak ada migration database karena field `Tenant.disabledModules` sudah ada.
- `Superadmin` auth sudah tersedia sebelumnya, jadi update ini memakai jalur superadmin yang sudah ada.
