# 2026-07-05 19:34 - Codex Update

## Ringkasan

Menambahkan fondasi fitur **QRIS Dinamis Manual** untuk POS dan memperbaiki
mode offline agar halaman yang sudah pernah dibuka bisa tampil lagi saat
koneksi terputus.

## Perubahan

- Menambahkan utility `buildDynamicQris()` untuk mengubah payload QRIS statis
  menjadi QRIS nominal transaksi.
- Algoritma QRIS:
  - hapus tag `54` lama jika ada,
  - sisipkan tag `54` baru berdasarkan total transaksi,
  - hitung ulang CRC-16 CCITT pada tag `63`.
- Menambahkan tampilan QRIS dinamis di modal pembayaran POS saat metode bayar
  `QRIS` dipilih.
- Menambahkan input placeholder "QRIS statis usaha" di modal pembayaran.
- QRIS statis sementara disimpan di `localStorage` device kasir agar bisa
  dipakai lagi saat offline.
- Tombol selesai untuk QRIS akan nonaktif jika QRIS statis kosong/tidak valid
  atau QR dinamis gagal dibuat.
- Memperbarui `public/sw.js`:
  - bump cache version ke `altora-v2`,
  - menambahkan cache halaman navigasi sebagai fallback offline,
  - halaman yang pernah dibuka akan dicoba ditampilkan dari cache saat offline.

## Catatan

- QRIS statis asli belum dimasukkan karena menunggu contoh QRIS dari owner.
- Fitur ini belum melakukan verifikasi pembayaran otomatis. Kasir tetap harus
  cek notifikasi/mutasi lalu menekan tombol selesai secara manual.
- Penyimpanan QRIS statis saat ini masih per-device via `localStorage`; versi
  production yang lebih rapi sebaiknya dipindah ke pengaturan outlet/tenant.
- Belum ada commit atau push.
