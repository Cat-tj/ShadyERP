# 2026-07-06 15:05 - Codex

## Konteks
- User melaporkan UI denah meja sulit terbaca karena label meja memakai lingkaran biru kecil.
- User meminta warna meja pastel dengan teks hitam, lebih hemat ruang, dan kursi otomatis mengikuti kapasitas meja.

## Yang Dikerjakan
- Mengubah `TableVisual` agar label meja langsung berada di badan meja, bukan badge lingkaran biru.
- Mengganti status warna meja menjadi pastel dengan teks hitam:
  - Kosong: biru pastel.
  - Baru pesan: amber pastel.
  - Sedang makan/dimasak: slate pastel.
  - Siap/tagihan: hijau pastel.
- Membuat kursi otomatis mengikuti kapasitas:
  - Meja panjang membagi kursi ke sisi atas, bawah, kiri, dan kanan.
  - Meja kotak membagi kursi merata di empat sisi.
  - Meja bulat menaruh kursi melingkar mengikuti kapasitas.
- Kapasitas besar dibatasi render visualnya sampai 32 kursi agar layout tetap stabil.
- Membersihkan warning lint lama pada modul meja/pesanan.
