# Update 2026-07-22 14:00 - Codex

## Scope

Menyamakan navigasi desktop semua tenant Altora agar tidak ada sektor yang
kembali memakai topbar desktop hanya karena `accountingMode = SIMPLE`.

## Changes

- `SimpleShell` sekarang memakai sidebar desktop yang setara dengan shell mode
  Advanced.
- Sidebar Simple menampilkan Ringkasan, Kasir, Uang, Data, Lainnya, serta
  Pengaturan untuk Owner.
- Submenu Pengaturan menggunakan `SettingsSidebarNav` yang sama untuk mode
  Simple dan Advanced.
- Tenant meneruskan `disabledModules` ke `SimpleShell`; modul nonaktif tetap
  tersembunyi di setiap sidebar.
- Tablet dan HP tetap memakai topbar ringkas dan bottom navigation agar ruang
  POS tidak terjepit.

## Affected Sectors

Cafe, Toko, Laundry, Counter, Pabrik, Company, dan semua tenant lain yang
menggunakan shell Simple atau Advanced.
