# Update 2026-07-22 14:45 - Codex

## Branding Auth per Subdomain

### Masalah

Halaman `/login` dan `/register` memakai logo dan aksen Altora global untuk
semua host, meskipun landing page sudah mengenali vertical melalui subdomain.

### Perubahan

- Menambahkan resolver vertical berbasis header `x-altora-vertical` yang sudah
  diisi proxy dari hostname.
- Login dan register sekarang memakai simbol, aksen warna, judul, serta copy
  sesuai vertical aktif.
- Metadata title dan favicon halaman auth juga menggunakan simbol vertical.

### Contoh

- `cafe.altora.my.id/login` memakai simbol dan warna Altora Cafe.
- `pabrik.altora.my.id/register` memakai simbol dan warna Altora Pabrik.
- `altora.my.id/login` tetap memakai simbol Altora utama.
