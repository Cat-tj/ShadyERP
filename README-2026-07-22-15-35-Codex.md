# Update 2026-07-22 15:35 - Codex

## Perbaikan Login Teams

### Root Cause

VPS menjalankan dua checkout yang berbeda:

- `/home/ubuntu/ShadyERP` melalui PM2 `shadyerp` pada port `3002`.
- `/home/altora/ShadyERP` melalui `altora-main.service` pada port `3000`.

Caddy mengarahkan domain publik ke port `3000`. Akun demo pertama sempat
dibuat pada checkout port `3002`, sehingga tidak terlihat oleh login publik.

### Perbaikan Operasional

- Akun Teams demo diprovision ulang pada database dari
  `/home/altora/ShadyERP`.
- Verifikasi memastikan password hash cocok, user dan tenant aktif, role
  `OWNER`, serta satu akses outlet tersedia.

### Perbaikan UI

- `src/components/login-form.tsx` sekarang memiliki tombol
  `Tampilkan`/`Sembunyikan` kata sandi.
- Komponen tersebut dipakai bersama oleh semua halaman login Altora; host hanya
  menentukan branding dan copy, bukan mekanisme autentikasi berbeda.

### Catatan Infrastruktur

Checkout port `3002` bukan upstream publik. Jangan membuat akun produksi dari
`/home/ubuntu/ShadyERP`; gunakan service aktif `altora-main.service` atau
alur register publik.
